- Start Kafka
Navigate to the folder where your docker-compose.yml file is located:

cd [your-folder-path]

Run the command docker-compose up in the terminal. 

This will start both Kafka and Zookeeper containers based on the configuration in your Docker Compose file.

-Verify Containers are Running
Check if the containers are running by opening another terminal and executing the command:

docker ps

You should see containers for both Kafka and Zookeeper listed as running.

-List All Topics
To see all existing topics, use:

kafka-topics.sh --list --bootstrap-server localhost:9092

Example topics:
handledorders
orderproject

- Create a Topic
To create a new topic, run the following command:

kafka-topics.sh --create --topic your_topic --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1

- Describe a Topic
To describe a specific topic, use:

kafka-topics.sh --zookeeper zookeeper:2181 \
  --describe --topic orders


- Delete a Topic
To delete a topic, run:

kafka-topics.sh --zookeeper zookeeper:2181 \
  --delete --topic orders

- Consume Messages from Beginning (e.g., handled_orders topic)
To consume messages from the beginning of a topic, run:

kafka-console-consumer.sh --bootstrap-server localhost:9092 --topic handled_orders --from-beginning
